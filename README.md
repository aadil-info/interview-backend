# 🤖 AI Interview Platform

An advanced AI-powered Interview Preparation Platform built using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript. This platform helps users prepare for technical interviews, aptitude rounds, and HR questions with AI feedback, mock interviews, and performance analytics.

---

# 🚀 Features

## 👤 Authentication
- User Signup & Login
- JWT Authentication
- Secure Password Hashing

## 🎯 Mock Interview System
- Technical Interview Questions
- HR Round Questions
- Aptitude Test Module
- Random Question Generator
- Difficulty Levels

## 🤖 AI Feedback
- Answer Analysis
- AI-Based Suggestions
- Performance Score
- Improvement Recommendations

## 📊 Dashboard & Analytics
- Interview Progress Tracking
- Performance Reports
- Charts & Statistics
- Leaderboard System

## 📄 Resume Analyzer
- Resume Upload
- ATS Score Checker
- Resume Improvement Suggestions

## ⚡ Additional Features
- Dark/Light Mode
- Responsive UI
- Admin Panel
- Notes & Bookmarking
- Real-Time Progress Tracking
